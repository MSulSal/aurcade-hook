# Aurcade Hook 4 - Full Production Site Example

Hook 4 is a full end-product site sample, not a billboard.

## Coverage

- `index.html`: Home (Game of the Day, Newest World Records, Upcoming Events, Recent Scores)
- `leaderboards.html`: Newest Scores and ranking views
- `games.html`: Browse Games with search and filters
- `venues.html`: Browse Locations with search and type filters
- `events.html`: Events schedule and recent event results
- `community.html`: Public Forums + score submission and feature voting
- `resources.html`: About Us + project/resource sections

## Shared Files

- `styles.css`: 90s arcade-escape visual system
- `main.js`: active nav, updated timestamp, filters, search, and vote interactions

## Content Note

Text labels and section wording are aligned to current Aurcade language and page structure.

## Run

```powershell
python -m http.server 8080
```

Then open:

- `http://localhost:8080/hook-4/`
