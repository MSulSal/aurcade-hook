# Aurcade Hook Workspace

This repo is organized into multiple concept folders (`hook-1`, `hook-2`, etc.).

## Start Here

1. Read [SOURCE_OF_TRUTH.md](./SOURCE_OF_TRUTH.md)
2. Open the hook folder you want to run or edit

## Current Hooks

- `hook-1`: first complete concept
- `hook-2`: retro portal concept (more nostalgic, less modern)
- `hook-3`: serious sample site using hook-2 aesthetic as the base
- `hook-4`: full production-site example with Aurcade-aligned structure and text
