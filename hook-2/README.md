# Aurcade Hook 2

Hook 2 pushes harder into a classic retro web portal style.

## Concept Thesis

Make Aurcade feel like the home screen of an old-school arcade network:
dense, link-rich, score-first, and unmistakably nostalgic.

## Visual Direction

- More retro, less modern polish
- Directory-style layout with side rails
- Beveled panels, classic table UI, bitmap-like headings
- Scrolling network feed and blinking status cues

## Files

- `index.html`
- `styles.css`
- `main.js`

## Run

Open `index.html` directly or run:

```powershell
python -m http.server 8080
```

Then open `http://localhost:8080/hook-2/`.
